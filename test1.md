# マークダウンの書き方基礎

マークダウンは簡潔で読みやすい記法です。以下に基本的な使い方を紹介します。

## 見出し

```
# 見出し1
## 見出し2
### 見出し3
```

## リスト

### 箇条書きリスト
- アイテム1
- アイテム2
  - ネストしたアイテム

### 番号付きリスト
1. 最初のアイテム
2. 2番目のアイテム

## 強調

*イタリック* や **太字** も簡単に書けます。
