# はじめての投稿

こんにちは！これは Jekyll で作成した最初のブログ投稿です。Jekyll は静的サイトジェネレーターで、マークダウンで書いた文章を HTML に変換してくれます。

## Jekyll の特徴

- マークダウンで簡単に記事が書ける
- GitHub Pages との相性が良い
- テーマのカスタマイズが容易
- プラグインで機能拡張が可能

### コードの挿入例

```ruby
def hello_world
  puts "Hello, Jekyll!"
end
```

詳細は [Jekyll の公式サイト](https://jekyllrb.com/) をご覧ください。
